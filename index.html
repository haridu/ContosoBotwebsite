Contoso bot

A Bot that help you with currency exchange needs

Functionlities

No. Functions Description
1. Identify New Zealand currency notes Using Custom vision AI bot is able to Identify New Zealand currency notes with not New Zealnd currncy notes
2. Intent recognition Using Luis AI bot is able to identify the intentions os the user and provides available required functionalities.
3. Identify users inputed language Using Text analysis AI bot is able to identify user inputted language,This mainly used for welcome intent and when user provide non English input bot will reply with an messege mentioning that bot currently only supports English language.
4. Get Exchange rate between 2 currencies Using Fixer.io Bot can get the exchange rate between 2 currencies
5. Get Exchange rate based on user request currency Using Fixer.io Bot can provide exchange rates based on users preference
6. Add,Delete,Get currencies with favourite currency list Bot provides personal favorite currency list functionality to the user. Bot does this by calling to Azure Easy Tables
7. Login This functionality is available as stand alone function to demonstrate.Unstability of the functionality was the reason behind implementing it as a stand alone function.

Technology Used

  • Bot Framework with Node.js
  • Fixer.io
  • Custom Vision
  • Text analysis
  • Luis.ai
  • Azure Easy Tables
